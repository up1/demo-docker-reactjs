# Working with ReactJS + Docker

### Step 1 :: Build 
```
// Install dependencies from file `package.json`
$yarn

// Build process
$yarn build
```
Created files in folder `/build`
* HTML
* Image
* JavaScript (JS)
* Stylesheet (CSS)

### Step 2 :: 
Deploy folder `/build` into web server (nginx)

For develoment mode ::
```
$yarn start
```
Open url=`http://localhost:3000/` in your browser
